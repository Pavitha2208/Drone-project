# AI-Powered Drone Operations Dashboard

This project is a React-based web dashboard designed to manage drone missions, task assignments, cargo operations, live monitoring, and AI-based alerts.  
It is developed as an academic and prototype project focusing on automation and operational efficiency.

## Features
- User login and signup
- User and operator dashboards
- Task and cargo management
- Live video monitoring
- AI-based alerts
- Mission control and task tracking
- Task history and reports

## Tech Stack
- React.js
- React Router
- Context API
- CSS
- Git and GitHub

## How to Run
1. Clone the repository
2. Run npm install
3. Run npm start
4. Open http://localhost:3000

## Purpose
This project demonstrates an AI-assisted drone operation workflow and is intended for educational and learning purposes.
